OhEmbedr is an easy to use ruby library for using OEmbed services in your applications. See oembed.com for information on the OEmbed standard. This library supports the following services by default:

  • YouTube
  • Vimeo
  • Flickr
  • Qik
  • Revision 3
  • Viddler
  • Hulu

See the OhEmbedr class page for usage details.
